Refrigerator Not Cooling

Error codes: E01, E02, E03

Common causes include dirty condenser coils, faulty thermostat, or compressor issues. Check the following:

  • Clean condenser coils behind the refrigerator
  • Check temperature settings (should be 37-40°F)
  • Ensure door seals are tight and clean
  • Listen for compressor operation sounds

Washing Machine Not Draining

Error codes: F21, E21, dE

Usually caused by clogged drain hose, pump filter, or faulty drain pump. Try these steps:

  • Check and clean the drain pump filter
  • Inspect drain hose for kinks or clogs
  • Run a cleaning cycle with vinegar
  • Check if the drain pump is operating

AC Not Cooling Properly

Error codes: E1, E4, H4

Low refrigerant, dirty filters, or compressor issues can cause poor cooling. Check these:

  • Clean or replace air filters regularly
  • Check outdoor unit for debris
  • Ensure thermostat is set correctly
  • Listen for unusual sounds from the unit

TV Screen Black But Sound Works

Error codes: No display, backlight failure

Backlight failure or T-con board issues are common causes. Try these troubleshooting steps:

  • Shine a flashlight on the screen to check for faint image
  • Check all cable connections
  • Try a different input source
  • Power cycle the TV (unplug for 60 seconds)

Microwave Not Heating

Error codes: F3, F4, H98

Magnetron failure, high voltage diode, or capacitor issues can cause no heating. Safety first:

  • Ensure the door closes properly
  • Check that the turntable rotates
  • Listen for normal operation sounds
  • ⚠️ High voltage components - professional repair recommended

Dishwasher Not Cleaning Dishes

Error codes: E1, E2, E3

Clogged spray arms, dirty filters, or water supply issues can cause poor cleaning. Check these:

  • Clean the spray arms and remove debris
  • Check and clean the filter
  • Ensure proper water temperature (120°F)
  • Use the correct amount of detergent
